BMOS Documentation.

Learn how to download, install, boot, configure, and use BMOS.

Requirements

You need a supported device or environment, storage media, and a way to flash or install BMOS.

Recommended:
• 16GB+ storage
• Keyboard and display
• Internet for updates
• BMOS image or installer

Download

Download the latest stable BMOS version from the downloads page.

BMOS-1.2-Stable.img.zip

Install

Flash BMOS to a drive or install it in your preferred supported environment.

1. Download BMOS
2. Extract the image
3. Flash or install
4. Boot the system

On first boot, BMOS may create settings, load packages, and start setup.

Troubleshooting

If BMOS does not boot, reflash the image, check your storage, and confirm your environment is supported.

Common fixes:
• Reflash BMOS
• Check storage
• Restart device
• Verify package files
• Check internet connection
