About BMOS

Built for developers.

BMOS is a lightweight terminal operating environment made for Raspberry Pi, learning, experimenting, customization, and building your own OS-like command-line experience.

Download BMOS Read Docs

What BMOS is.

BMOS is not trying to replace Linux. It is built on top of a lightweight Linux environment and gives users a clean, custom shell for learning, experimenting, and expanding through packages.

Minimal

BMOS keeps the interface simple and focused, giving you a fast terminal-first experience without unnecessary clutter.

Expandable

BMOS can grow with packages, repositories, updates, settings, and future internet-connected tools.

Educational

BMOS helps users learn Linux, Raspberry Pi, Python, shell design, and how software systems work behind the scenes.

Why BMOS exists.

BMOS was created to make experimenting with a custom operating system feel approachable. Instead of starting with something huge and complicated, BMOS gives you a smaller environment that you can understand, edit, and improve over time.

It is perfect for turning a Raspberry Pi into a personal terminal, learning how package systems work, or building your own command tools from scratch. 

BMOS Philosophy

• Fast startup
• Minimal interface
• Custom shell
• Package support
• Raspberry Pi focused
• Built for learning

BMOS is small on purpose.

Who BMOS is for.

BMOS is designed for anyone who wants a simple system they can experiment with, break, fix, customize, and actually understand.

Students

Learn real command-line concepts, file paths, permissions, packages, updates, and system structure.

Makers

Use BMOS as a base for Raspberry Pi projects, custom hardware builds, terminals, and small utility systems.

Developers

Build commands, package tools, improve the shell, and shape BMOS into your own lightweight environment.
