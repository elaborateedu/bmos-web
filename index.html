A terminal OS designed for experimentation.

BMOS combines the flexibility of Linux with a clean, focused command-line environment built specifically for learning, customization, and Raspberry Pi projects.

Lightweight

BMOS is optimized for Raspberry Pi systems, keeping startup times fast and resource usage low.

Package Based

Add functionality through packages and repositories, allowing BMOS to grow over time.

Educational

Learn Linux concepts, shell environments, Python scripting, and system design through experimentation.

Built for Raspberry Pi.

BMOS was designed around Raspberry Pi hardware, making it perfect for DIY projects, portable systems, retro terminals, and lightweight development environments.
